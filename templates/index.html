Social Media Image Optimizer API

Welcome to the Social Media Image Optimizer API. This service provides endpoints to optimize images for various social media platforms and generate platform-specific captions.

Available Endpoints

GET /platforms/

Returns a list of all supported social media platforms and their image dimensions.

Example Response:

{
  "platforms": {
    "instagram": {
      "name": "Instagram",
      "size": "1080x1080"
    },
    "twitter": {
      "name": "Twitter",
      "size": "1600x900"
    },
    ...
  }
}

POST /process-image/

Processes an image for a specific social media platform, with options for enhancement and smart cropping.

Parameters:

Name Type Required Description
file File Yes The image file to process
platform String Yes Target social media platform (e.g., instagram, twitter)
format String No Output image format (JPEG or PNG, default: JPEG)
smart_crop_enabled Boolean No Whether to use AI-powered smart cropping (default: false)
enhance_image_enabled Boolean No Whether to enhance image quality (default: false)
confidence Float No Detection confidence threshold for smart cropping (default: 0.3)

Example Response:

{
  "message": "Processing complete",
  "optimized_image": "/output_images/optimized_123e4567-e89b-12d3-a456-426614174000_image.jpg",
  "caption_text": "A beautiful sunset over the mountains #photooftheday #instagood #instamood",
  "platform": "instagram",
  "size": [1080, 1080],
  "enhanced": true,
  "smart_cropped": true
}

Using the API with the React Frontend

This API is designed to work with the provided React frontend. To use the frontend:

  1. Start the API server with uvicorn app:app --reload
  2. Navigate to the React app directory and run npm start
  3. Access the frontend at http://localhost:3000
